Projeto desenvolvido para o curso de dados da Cesar School.

Projeto1: Projeto desenvolvido com a tecnologia Python, com as bibliotecas beautifulsoup e pandas para fazer a extração e leitura de dados do html. Também foi utilizado Docker para criar imagem e rodar a mesma na nuvem.


Projeto2: Consiste em tratar os dados extraídos no projeto 1, foi utilizado a linguagem Python com a biblioteca Pandas para ler e tratar os dados necessarios.
